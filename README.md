# 🚀 AstraHost PteroMaster Installer

[![License](https://img.shields.io/badge/license-private-red.svg)](https://github.com/Danimaru-ze/AstraHost-Installer)
[![Developer](https://img.shields.io/badge/developer-@danimaruse-blue.svg)](https://t.me/danimaruse)

**AstraHost PteroMaster Installer** adalah toolkit otomasi profesional yang dirancang khusus untuk memudahkan pengelolaan, instalasi tema, dan optimasi keamanan pada Pterodactyl Panel.

---

## 🌟 Fitur Utama

- **Premium Theme Installer**: Instalasi otomasi untuk tema Stellar, Noobee (Latest), dan Enigma.
- **Node & Location Creator**: Membuat Node dan Lokasi langsung dari terminal tanpa masuk ke Dashboard.
- **Security Protection**: Fitur tambahan untuk mengamankan file sistem panel Anda.
- **Automated Repair**: Memulihkan panel ke kondisi default jika terjadi kesalahan konfigurasi.
- **Node.js 22 Support**: Mendukung pembangunan (build) tema pada Pterodactyl panel versi terbaru.

---

## 🛠️ Cara Penggunaan

Untuk menjalankan installer ini di VPS Anda, cukup gunakan perintah satu baris (one-liner) berikut:

```bash
bash <(curl -s https://raw.githubusercontent.com/Danimaru-ze/AstraHost-Installer/main/install.sh)
```

> [!IMPORTANT]
> Pastikan Anda memiliki **Access Token** yang valid untuk menggunakan toolkit ini.
